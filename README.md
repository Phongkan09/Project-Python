# Project-Python
โปรเจกต์รวบรวมงานฝึก Python ที่สร้างขึ้นเพื่อพัฒนาทักษะการเขียนโปรแกรมจากพื้นฐานถึงระดับกลาง โดยเน้นการฝึกคิดเชิงตรรกะ การใช้คำสั่งควบคุมการทำงาน และการแก้ปัญหาแบบ step by step

![VS Code](https://img.shields.io/badge/VS%20Code-007ACC.svg?style=for-the-badge&logo=visualstudiocode&logoColor=white)

โฟลเดอร์นี้เป็นพื้นที่เก็บผลงานฝึกเขียนโปรแกรม Python ของผู้เรียน โดยทุกไฟล์ล้วนมีวัตถุประสงค์เพื่อฝึกฝนทักษะที่สำคัญในชีวิตการพัฒนาโปรแกรมอย่างเป็นระบบ

## 📂 รายชื่อโปรเจกต์

| ชื่อไฟล์ | คำอธิบาย |
| :--- | :--- |
| [**CLI Expense Tracker.py**](./CLI%20Expense%20Tracker.py) | โปรแกรมจัดการรายรับ-รายจ่ายแบบ CLI ที่มีระบบสมัครสมาชิกและบันทึกประวัติการทำรายการ |
| [**Arithmetic Formatter Project.py**](./Arithmetic%20Formatter%20Project.py) | โปรแกรมฝึกจัดรูปแบบคำตอบทางคณิตศาสตร์ให้อ่านง่ายและสวยงาม |
| [**Budget App Project.py**](./Budget%20App%20Project.py) | โปรแกรมฝึกสร้างแอปจัดการงบประมาณพื้นฐาน |
| [**Polygon Area Calculator Project.py**](./Polygon%20Area%20Calculator%20Project.py) | โปรแกรมคำนวณพื้นที่ของรูปหลายเหลี่ยมตามเงื่อนไขที่กำหนด |
| [**Probability Calculator Project.py**](./Probability%20Calculator%20Project.py) | โปรแกรมฝึกคำนวณความน่าจะเป็นแบบง่าย ๆ |
| [**Time Calculator Project.py**](./Time%20Calculator%20Project.py) | โปรแกรมคำนวณเวลาและแปลงค่าต่าง ๆ ที่เกี่ยวข้องกับเวลา |

## ✨ จุดประสงค์ของโปรเจกต์

โปรเจกต์ในโฟลเดอร์นี้ถูกสร้างขึ้นเพื่อฝึกทักษะดังต่อไปนี้:
- การคิดเชิงตรรกะและแก้ปัญหา
- การเขียนโปรแกรมแบบมีโครงสร้างและอ่านง่าย
- การใช้ตัวแปร ฟังก์ชัน และคำสั่งควบคุมการทำงาน
- การรับข้อมูลจากผู้ใช้และแสดงผลลัพธ์อย่างเหมาะสม
- การฝึกเขียนโค้ดด้วยภาษา Python ในรูปแบบที่เข้าใจง่าย

## 🛠️ วิธีการรันโปรเจกต์

1. เปิด Terminal หรือ Command Prompt
2. เข้าไปยังโฟลเดอร์ Project-Python
   ```bash
   cd Project-Python
   ```
3. รันไฟล์ Python ที่ต้องการโดยใช้คำสั่งดังนี้
   ```bash
   python "CLI Expense Tracker.py"
   ```
   หรือรันไฟล์อื่น ๆ เช่น
   ```bash
   python "Arithmetic Formatter Project.py"
   ```
