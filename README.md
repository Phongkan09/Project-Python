## ✨ รายละเอียดแต่ละโปรเจกต์

### 1. CLI Expense Tracker
โปรเจกต์นี้เป็นแอปพลิเคชันจัดการงบประมาณแบบข้อความที่ทำงานใน Terminal โดยผู้ใช้สามารถ:
- สมัครสมาชิกและเข้าสู่ระบบด้วยบัญชีของตัวเอง
- เพิ่มรายการรายรับและรายจ่าย
- ดูยอดคงเหลือของบัญชีปัจจุบัน
- ดูประวัติการทำรายการทั้งหมด
- บันทึกข้อมูลลงในไฟล์เพื่อเก็บข้อมูลไว้ใช้งานต่อไป

โปรเจกต์นี้ช่วยฝึกเรื่องสำคัญมากมาย เช่น:
- การใช้ Class และ Object
- การอ่าน/เขียนไฟล์ใน Python
- การตรวจสอบข้อมูลจากผู้ใช้
- การสร้างเมนูแบบโต้ตอบด้วยคำสั่งใน Terminal
- การจัดการข้อมูลแบบพื้นฐานของแอปพลิเคชัน

### 2. freecodecamp Projects
โฟลเดอร์นี้ประกอบด้วยโปรเจกต์ฝึกฝนที่มาจากหลักสูตร freeCodeCamp โดยเน้นการใช้ Python ในการแก้ปัญหาจริง ๆ และฝึกฝนทักษะต่อไปนี้:
- การเขียนฟังก์ชันและใช้ตรรกะในการแก้ปัญหา
- การจัดรูปแบบผลลัพธ์ให้อ่านง่าย
- การคำนวณและประมวลผลข้อมูลเชิงตัวเลข
- การสร้างโปรแกรมที่มีโครงสร้างชัดเจนและเข้าใจง่าย

## 🧠 ทักษะที่ฝึกฝนจากโปรเจกต์นี้

- การคิดเชิงวิเคราะห์และแก้ปัญหา
- การเขียนโค้ดแบบเป็นขั้นตอนและมีโครงสร้าง
- การใช้ตัวแปร ค่าตัวแปรและประเภทข้อมูลต่าง ๆ
- การใช้คำสั่งควบคุมการไหลของโปรแกรม เช่น if, for, while
- การสร้างฟังก์ชันและจัดกลุ่มโค้ดให้อยู่ในส่วนที่เหมาะสม
- การจัดการไฟล์และข้อมูลจากผู้ใช้
- การสร้างโปรแกรมที่โต้ตอบกับผู้ใช้ผ่าน Terminal

## 🛠️ วิธีการใช้งาน (How to Run)

1. Clone โฟลเดอร์นี้ลงในเครื่องของคุณ:
   ```bash
   git clone https://github.com/Phongkan09/Practice-Project-Web.git
   ```

2. เข้าไปยังโฟลเดอร์ Project-Python:
   ```bash
   cd Project-Python
   ```

3. รันไฟล์ Python ที่ต้องการ

   สำหรับโปรเจกต์ CLI Expense Tracker:
   ```bash
   python "CLI Expense Tracker/CLI Expense Tracker.py"
   ```

   หากต้องการรันโปรเจกต์อื่น ๆ ในโฟลเดอร์ freecodecamp:
   ```bash
   python "freecodecamp/ชื่อไฟล์.py"
   ```

4. ถ้าต้องการใช้ฟีเจอร์บางอย่างใน CLI Expense Tracker อาจต้องติดตั้งไลบรารีเพิ่มเติม เช่น:
   ```bash
   pip install prompt_toolkit
   ```
